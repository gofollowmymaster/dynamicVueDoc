# 预定义配置信息

TIP

为了简化配置,快速上手.Dy-Vue 预定义了大量组建/操作配置可以满足大部分常规需求,不过您也可以自由的定义自己的配置,使用appendToPreset方法将会吧您自定义的配置与预定义配置深度合并

配置名 含义 默认值
searchOption 搜索配置选项 searchOption
tableOption 表格配置选项 tableOption
dynamicFormOption 动态表单配置 DynamicFormOption
pagination 分页配置 pagination
treeOption tree配置 pagination
formOption 表单配置 formOption
popTipOptions 操作气泡提醒 pagination
msgBoxTipOptions 操作消息提醒 msgBoxTipOptions
drawerProperties 默认抽屉属性 drawerProperties
dialogProperties 默认弹窗属性 dialogProperties

| actionTip | 操作提醒方式 | 可选 pop msg | | baseFormSectionName | 基础表单区块信息名 | 基础信息 | | btnSize | 按钮尺寸 | small | | isDebug | 是否调试模式（快速填充表单） | true | | mainKey | 数据主键 | id | | defaultDialogMode | 弹窗类型 | dialog drawer router |
